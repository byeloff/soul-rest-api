# Rest API Simples com Autenticação JWT

[![N|Solid](http://gabrielsousa.com.br/assets/images/logo.png)](http://gabrielsousa.com.br)

  - Lumen
  - CRUD de Usuários
  - CRUD de Artigos
  - Migrations 
  - Exception tratada
  - Middleware para o JWT

# ROTAS

```sh
$ GET /api/users
$ POST /api/users
$ PUT /api/users/{id}
$ DELETE /api/users/{id}
```

```sh
$ GET /api/posts
$ POST /api/posts
$ PUT /api/posts/{id}
$ DELETE /api/posts/{id}
```
